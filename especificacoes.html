Câmera Infravermelho

Câmera que capta luz infravermelho

Flash Infravermelho

Preenche o rosto do usuário com luz infravermelha não vista pelo olho humano. Essa luz serve de “flash” para a câmera infravermelha. Isso explica o reconhecimento facial mesmo no escuro.

Sensor de proximidade

As câmeras com sensor de profundidade e o mapeamento facial preciso possibilitam criar efeitos de iluminação com qualidade de estúdio. Tambem usado para que o sistema saiba quando um usuário está próximo o suficiente para ativar o alto-falante para chamadas.

Sensor de Luz Ambiente

Um sensor de luz ambiente ajuda o sistema a definir níveis de luz de saída. O novo recurso do modo Retrato cria efeitos de iluminação profissional.

Alto Falante

Alto Falante ativado em conjunto com o alto falante inferior. Juntos, eles proporcionam som estéreo ao usuário.
